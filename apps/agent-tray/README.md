# @voicechat/agent-tray — компаньон-агент с иконкой в трее (macOS)

Electron-приложение: выполняет shell-команды голосового ассистента на этой машине,
показывает иконку в menu bar со статусом, ведёт журнал команд и позволяет
остановить/выйти из меню трея. Ядро подключения переиспользуется из `apps/agent`.

## Запуск (для пользователя)

1. В веб-клиенте: Настройки → Агент → «Добавить машину» → «⬇ Скачать приложение» и
   «Скопировать строку подключения».
2. Установите `.dmg` (без подписи → ПКМ по приложению → «Открыть»).
3. При первом запуске вставьте строку подключения → в menu bar появится иконка,
   статус «● В сети — <имя машины>».
4. Меню трея: «Показать журнал», «Остановить»/«Возобновить», «Настройки…», «Выход».

## Разработка

```bash
npm --prefix apps/agent-tray install
npm --prefix apps/agent-tray run dev        # запуск в dev
npm --prefix apps/agent-tray run dist       # сборка .dmg → release/
npm --prefix apps/agent-tray run dist:dir   # быстрый .app без dmg
```

Приложение вне npm workspaces (свой `node_modules` с Electron), как `apps/desktop`.
Единственная рантайм-зависимость — `ws`; ядро агента и общий код бандлятся из
исходников (`@agent`, `@shared`).
