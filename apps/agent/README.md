# @voicechat/agent — компаньон-агент

Консольное приложение для машины пользователя: выполняет shell-команды голосового
ассистента локально (а не на сервере). Подключается к серверу по WebSocket и
авторизуется токеном.

## Запуск

1. В веб-клиенте: Настройки → Агент → «Машины» → «Добавить машину» — скопируйте токен.
2. На своей машине:

```bash
npx tsx apps/agent/src/index.ts --server ws://<сервер>:8787/agent --token <токен>
```

Или через переменные окружения `VC_AGENT_SERVER` и `VC_AGENT_TOKEN`:

```bash
VC_AGENT_SERVER=ws://192.168.1.10:8787/agent VC_AGENT_TOKEN=abc… npm start --workspace @voicechat/agent
```

3. В настройках выберите машину в «Где выполнять команды».

Агент сам переподключается при обрыве связи (backoff до 30 с). Токен отзывается
удалением машины в настройках.
